	\subsection*{Part 1: Totally Total?}
	Give reductions showing that $ \bar{HP} \leq \{M : M\text{ is total}\} $ and $ \bar{HP} \leq \{M : M\text{ is not total}\} $. Briefly argue that each reduction is correct.
	
	\linesep
	
	We will first show that using a total Turing machine with the language $ \{M : M\text{ is total}\}  $, we can create a total Turing machine for $ \bar{HP} $.
	\begin{algorithmic}
			\IF{$ N $ halts on $ x $}
			\RETURN{ $ N' $ such that $ N' $ loops on all inputs immediately (never enters accept or reject state)}
			\ELSE
			\RETURN $ N' $ such that $ N' $ accepts all inputs immediately.
			\ENDIF
		\end{algorithmic}
	\begin{proof}
		If $ N $ halts on $ x $ then $ N' $ is not total, so $ M^* $ rejects $ N' $, and therefore rejects $ N\#x $. If $ N $ loops on $ x $ then $ N' $ is total, so $ M^* $ accepts $ N' $, and therefore accepts $ N\#x $. So $ N $  loops on $ x $ if and only if $ N' $ is total.\\
		
		Now we will show that using a total Turing machine with the language $ \{M : M\text{ is not total}\}  $, we can create a total Turing machine for $ \bar{HP} $.
		\begin{algorithmic}
			\IF{$ N $ halts on $ x $}
			\RETURN{$ N' $ such that $ N' $ accepts all inputs immediately.}
			\ELSE
			\RETURN $ N' =N$
			\ENDIF
		\end{algorithmic}
		\begin{proof}
			If $ N $ halts on $ x $ then $ N' $ is total, so $ M^* $ rejects  $ N' $, and therefore rejects $ N\#x $. If $ N $ loops on $ x $ then $ N' $ is $ N $ which loops on $ x $ and hence is not total, so $ M^* $ accepts $ N' $, and therefore accepts $ N\#x $. So $ N $ loops on $ x $ if and only if $ N' $ is not total.
		\end{proof}
	\end{proof}
